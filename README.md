# Food-Ordering-Behaviour-and-Consumer-Trends-A-Structured-Analysis-of-Choices-and-Habits
##  Project Overview

This project focuses on analyzing food ordering behaviour and consumer trends using Tableau. The aim is to explore customer preferences, ordering patterns, delivery performance, and business insights from food delivery data. By analyzing various attributes, the project helps understand customer behaviour and supports data-driven decision-making.

> **Project Status:**  Work in Progress

---

##  Project Objectives

- Analyze customer ordering behaviour.
- Study food ordering trends across different cities.
- Understand customer preferences based on cuisine, meal type, and order value.
- Evaluate delivery performance and efficiency.
- Create interactive dashboards and visualizations using Tableau.

---

##  Dataset Features

The dataset includes attributes such as:

- Order ID
- Order Value
- Delivery Fee
- Cuisine
- Meal Type
- City
- Delivery Time
